Lecture notes

DOM --> Document Object model, Generated from the html

- Tree structtire with the document as root which is why document.write() does what it does

- Also why css works by traversing the tree, selecting elements and or their sub-elements

How to manipulate the DOM with javascript

The first step is to find the element document.document element

Useful ways to retrieve elements:
document.getElementsByTagName("p")
document.getElementsByClassName("some-class-name")
document.getElementById('element name')

It is useful to add something like El to the end of your variable names to keep track of what they are.



elements can be created via the dom not just through text:
1. get the element
ex : var targetEl = document.getElementById(...)
2. create new element
ex : var newEl = document.createElement("button")
.....newEl.innerHTML = "press me"
.....newEl.style.backgroundColour = 'red'
3. insert the created element to the parent
ex :targetEl.appendChild(newEl)
